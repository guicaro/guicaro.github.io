PARA JUGAR

  • La letra "i" hace que la rana se mueva hacia arriba

  • La letra "k" hace que se vaya para abajo

  • La letra "j" hace que se vaya para la izquierda

  • La letra "l" hace que se vaya para la derecha

  • Para empezar a jugar darle 2 clicks a la imagen de arriba ya que este cargado.

  • OBJETIVO: Hacer que 5 ranas lleguen al otro lado del río.
