Projects

I've done some projects including a few websites designed in HS, and a Digital Yearbook which I suggested and managed. I've included links to some of these web sites and a brief explanation of the project.

At the moment I am exploring the possibilites of having my cellphone (Sony Ericsson T610) connect to a server thourgh Bluetooth, in order to access Skype (VOIP software). Since the phone has JAVA capabilities, I am researching the possibilites of creating an application that will do this. The benefits, would be: communicating with other Skype users for free and also to regular telphone users, without using your monthly minutes assigned by your provider.

I will include a link to my University Registration Database project. It was implemented using IBM's DB2 UDB and Websphere Studio Application Developer. It uses JSP's and JDBC. However I am working on redesigning the desing, so that it complies with the 3NF standard.
