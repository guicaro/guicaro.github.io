Bienvenidos.

Este sitio web, lo que tiene son secciones de las identidades vistas en el curso de Matemáticas V (Fundamentales, dobles, cosientes y miscelanea). Basicamente es en lo que se concentra. El objetivo de este sitio es mostrar como se resuelven las identidades paso a paso, además en este sitio se incluye un juego matematico que es muy divertido.

Este sitio web de Identidades fué creado por el alumno de 5 ° A de Preaparatoria

Guillermo Cabrera R.

Este sitio fué creado como un proyecto de matemáticas para la semana cultural del Fray Luis de León. Teniendo como maestro al:

Prof. Juan Carlos Galvan Resendiz
